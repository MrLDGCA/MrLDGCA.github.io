Sudoku solver - Backtracking algorithm
A Python program that solve Sudoku puzzles using the Backtracking algorithm. This is a brute-force approach to solving the puzzle. The algorithm will progressively fill the blanks with numbers in ascending order. If the number doesn't violate the rules, the program will move onto the next blank. Else, the program will try the next value. Should all numbers fail, the program will go back to the previous blank and change that to the next value. This will repeat untill no blank spaces are left.
